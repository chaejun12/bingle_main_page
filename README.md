# 빙글이 (Binglee) — 메인 인터페이스

대리 돌봄(Proxy Care)으로 자연스러운 번아웃 해소를 돕는 웰니스 앱 **빙글이**의 메인 페이지 프로토타입입니다.
Finch(self-care pet)와 유사한 캐릭터 중심 데일리 케어 인터페이스로 구성했습니다.

## 세계관

빙글이는 **'행복을 만드는 얼음'** — 어느 날 아이스 아메리카노 컵 속에서 말을 걸어온 작은 얼음입니다.
얼음들은 태어나면 반드시 한 사람을 행복하게 만들어야 하며, 그러지 못하면 녹아서 물이 됩니다.
담당 인간의 상태에 따라 녹는 속도가 달라지기 때문에, 빙글이는 매일 '생존 매뉴얼'을 꺼내
사용자의 상태를 확인합니다.

> "네가 다시 웃을 수 있을 때까지, 나는 절대 녹지 않을 거야."

사용자가 자기 자신이 아닌 빙글이를 돌보는 이타적 행동을 통해 회복 루틴을 형성합니다.
번아웃 상태에서는 자신을 위한 동기가 고갈되지만, 타인을 향한 돌봄 행위는 동기를
유지할 수 있다는 가설에 기반합니다.

## 화면 구성

| 영역 | 설명 |
|------|------|
| 감정 온도 지수 | 12°C(소진) ~ 100°C(회복 완료) 게이지 — 상태 확인 결과로 설정되고 돌봄 활동으로 상승 |
| 냉장고 내부 무대 | 화면 대부분을 차지하는 냉장고 안 배경(내부 조명·선반·소품) · 빙글이(얼음 큐브 SVG) |
| 📝 상태 확인 | 스토리의 '생존 매뉴얼' — 최초 1회 이름 입력 후 KEDS 기반 3문항(5점 척도) 설문. 답변마다 빙글이의 얄미운 코멘트, 결과에 따라 감정 온도 설정 |
| 📋 돌봄 미션 팝업 | 산책 드로잉, 감사 편지, 명상, 수면 루틴, V-log — 체크 시 온도 상승 |
| 🎨 꾸미기 | 냉장고 안을 꾸미는 기능 자리 (준비 중 — `#decor-layer`에 아이템 배치 예정) |
| 🫧 얼음동굴 마을 | 하단 내비 '얼음동굴' 탭 — 내 동굴과 친구들의 동굴이 마을처럼 배치된 커뮤니티 |
| 하단 내비게이션 | 홈 · 냉장고 · 산책 · 얼음동굴 · 기록 |

## 얼음동굴 마을 (친구 커뮤니티)

- **마을 뷰**: 오로라가 흐르는 설원에 내 동굴(우편함 📮 배지)과 친구 동굴 4개가 배치.
  이글루마다 주인의 빙글이가 나와 있고, 이름표에 상태(휴식 중 🟢/바쁨 🟠)와
  나와 같은 휴식 시간(🕘) 여부가 표시됨. 동굴 클릭으로 상호작용
- **🎬 오늘의 V-log 모아보기 (셋로그 스타일)**: 하루가 마무리되면(21시 이후 마을 입장 시 자동)
  나와 친구들의 하루 V-log를 스토리 플레이어로 감상. 각 활동은 그 사람의 빙글이가
  귀엽게 대신 재현 — 일하면 빙글이가 노트북을 두드리고(💻 타닥타닥), 운동하면 점프,
  산책하면 뒤뚱뒤뚱 걷는 애니메이션. 자동 진행 + 좌우 탭 탐색 + 인물 탭 전환
- **💌 편지 쓰기**: 나와 같은 휴식 시간인 친구가 먼저 추천되고,
  "오늘 같이 영화 볼래? 🍿" 같은 한 줄 편지를 템플릿 칩 또는 직접 입력으로 전송.
  보낸 편지는 localStorage에 저장되어 내 동굴 우편함에서 확인 가능
- **친구 동굴 방문**: 친구의 빙글이가 지금 무엇을 대신 하고 있는지 보여주고,
  그 자리에서 V-log 보기 / 편지 쓰기로 이어짐
- **내 동굴**: 받은 편지(📮)와 보낸 편지 목록

## 인터랙션

- 첫 방문 시 빙글이가 이름을 묻고("앞으로 계속 같이 있을 건데, 너라고 부를 순 없으니까")
  이후 인사말에 이름을 반영 (localStorage 저장)
- 상태 확인 결과 감정 온도가 40°C 미만이면 **빙글이가 녹기 시작** — 물방울이 흐르고
  바닥에 물웅덩이가 생기며, 녹는 것에 대한 대사가 나옴
- 빙글이 터치 → 점프 + 하트 이펙트, 가끔 웃음을 기록 ("✏️ 기록: OO, 7시 42분. 아주 작게 웃음.")
- 미션 체크 → 온도 상승, 5개 완료 시 축하 연출
- 시간대별 인사말 자동 변경

## 앞으로 추가할 것

- **냉장고 꾸미기**: `#decor-layer` 레이어에 가구·조명·스티커 등 아이템 배치.
  선반 위 소품(`.prop[data-slot]`)이 교체 가능한 슬롯으로 준비되어 있음
- 산책 궤적 드로잉, 감사 편지 작성, V-log 등 미션별 상세 화면
