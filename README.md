# 빙글이 (Binglee) — 메인 인터페이스

대리 돌봄(Proxy Care)으로 자연스러운 번아웃 해소를 돕는 웰니스 앱 **빙글이**의 메인 페이지 프로토타입입니다.
Finch(self-care pet)와 유사한 캐릭터 중심 데일리 케어 인터페이스로 구성했습니다.

## 컨셉

사용자가 자기 자신이 아닌, 냉장고에 사는 캐릭터 **빙글이**를 돌보는 이타적 행동을 통해
회복 루틴을 형성합니다. 번아웃 상태에서는 자기 자신을 위한 동기가 고갈되지만,
타인을 향한 돌봄 행위는 동기를 유지할 수 있다는 가설에 기반합니다.

## 화면 구성

| 영역 | 설명 |
|------|------|
| 감정 온도 지수 | 12°C(소진) ~ 100°C(회복 완료) 게이지 — 돌봄 활동으로 상승 |
| 냉장고 무대 | 빙글이 캐릭터(SVG 애니메이션) · 말풍선 · 치유 식재료 선반 |
| 오늘의 돌봄 미션 | 산책 드로잉, 감사 편지, 명상, 수면 루틴, V-log — 5가지 솔루션 풀 기반 |
| 얼음동굴 | 커뮤니티 진입 배너 |
| 하단 내비게이션 | 홈 · 냉장고 · 산책 · 얼음동굴 · 기록 |

## 인터랙션

- 빙글이 터치 → 점프 + 하트 이펙트 + 말풍선 반응
- 치유 식재료 주기 → 음식 애니메이션 + 온도 상승
- 미션 체크 → 감정 온도 상승, 5개 완료 시 축하 토스트
- 시간대별 인사말 자동 변경
