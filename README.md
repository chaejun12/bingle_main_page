# 빙글이 (Binglee) — 메인 인터페이스

대리 돌봄(Proxy Care)으로 자연스러운 번아웃 해소를 돕는 웰니스 앱 **빙글이**의 메인 페이지 프로토타입입니다.
Finch(self-care pet)와 유사한 캐릭터 중심 데일리 케어 인터페이스로 구성했습니다.

## 세계관

빙글이는 **'행복을 만드는 얼음'** — 어느 날 아이스 아메리카노 컵 속에서 말을 걸어온 작은 얼음입니다.
얼음들은 태어나면 반드시 한 사람을 행복하게 만들어야 하며, 그러지 못하면 녹아서 물이 됩니다.
담당 인간의 상태에 따라 녹는 속도가 달라지기 때문에, 빙글이는 매일 '생존 매뉴얼'을 꺼내
사용자의 상태를 확인합니다.

> "네가 다시 웃을 수 있을 때까지, 나는 절대 녹지 않을 거야."

사용자가 자기 자신이 아닌 빙글이를 돌보는 이타적 행동을 통해 회복 루틴을 형성합니다.
번아웃 상태에서는 자신을 위한 동기가 고갈되지만, 타인을 향한 돌봄 행위는 동기를
유지할 수 있다는 가설에 기반합니다.

## 화면 구성

| 영역 | 설명 |
|------|------|
| 감정 온도 지수 | 12°C(소진) ~ 100°C(회복 완료) 게이지 — 상태 확인 결과로 설정되고 돌봄 활동으로 상승 |
| 냉장고 내부 무대 | 화면 대부분을 차지하는 냉장고 안 배경(내부 조명·선반·소품) · 빙글이(얼음 큐브 SVG) |
| 📝 상태 확인 | 스토리의 '생존 매뉴얼' — 최초 1회 이름 입력 후 KEDS 기반 3문항(5점 척도) 설문. 답변마다 빙글이의 얄미운 코멘트, 결과에 따라 감정 온도 설정 |
| 📋 돌봄 미션 팝업 | 산책 드로잉, 감사 편지, 명상, 수면 루틴, V-log — 체크 시 온도 상승 |
| 🎨 냉장고 꾸미기 | 벽 테마 6종 × 바닥 테마 6종을 독립적으로 골라 조합하는 팝업 (localStorage 저장) |
| 🫧 얼음동굴 마을 | 하단 내비 '얼음동굴' 탭 — 내 동굴과 친구들의 동굴이 마을처럼 배치된 커뮤니티 |
| 하단 내비게이션 | 홈 · 냉장고 · 산책 · 얼음동굴 · 기록 |

## 얼음동굴 마을 (친구 커뮤니티)

- **마을 뷰**: 오로라가 흐르는 설원에 내 동굴(우편함 📮 배지)과 친구 동굴 4개가 배치.
  이글루마다 주인의 빙글이가 나와 있고, 이름표에 상태(휴식 중 🟢/바쁨 🟠)와
  나와 같은 휴식 시간(🕘) 여부가 표시됨. 동굴 클릭으로 상호작용
- **🎬 오늘의 V-log 모아보기 (셋로그 스타일)**: 하루가 마무리되면(21시 이후 마을 입장 시 자동)
  나와 친구들의 하루 V-log를 스토리 플레이어로 감상. 각 활동은 그 사람의 빙글이가
  귀엽게 대신 재현 — 일하면 빙글이가 노트북을 두드리고(💻 타닥타닥), 운동하면 점프,
  산책하면 뒤뚱뒤뚱 걷는 애니메이션. 자동 진행 + 좌우 탭 탐색 + 인물 탭 전환
- **💌 편지 쓰기**: 나와 같은 휴식 시간인 친구가 먼저 추천되고,
  "오늘 같이 영화 볼래? 🍿" 같은 한 줄 편지를 템플릿 칩 또는 직접 입력으로 전송.
  보낸 편지는 localStorage에 저장되어 내 동굴 우편함에서 확인 가능
- **친구 동굴 방문**: 친구의 빙글이가 지금 무엇을 대신 하고 있는지 보여주고,
  그 자리에서 V-log 보기 / 편지 쓰기로 이어짐
- **내 동굴**: 받은 편지(📮)와 보낸 편지 목록

## 인터랙션

- 첫 방문 시 빙글이가 이름을 묻고("앞으로 계속 같이 있을 건데, 너라고 부를 순 없으니까")
  이후 인사말에 이름을 반영 (localStorage 저장)
- 상태 확인 결과 감정 온도가 40°C 미만이면 **빙글이가 녹기 시작** — 물방울이 흐르고
  바닥에 물웅덩이가 생기며, 녹는 것에 대한 대사가 나옴
- 빙글이 터치 → 점프 + 하트 이펙트, 가끔 웃음을 기록 ("✏️ 기록: OO, 7시 42분. 아주 작게 웃음.")
- 미션 체크 → 온도 상승, 5개 완료 시 축하 연출
- 시간대별 인사말 자동 변경

## 냉장고 꾸미기 (바닥 · 벽 테마)

우측 🎨 버튼을 누르면 벽/바닥을 각각 6개 테마 중에서 자유롭게 골라 조합할 수 있습니다.

| 테마 | 팔레트 | 분위기 |
|------|--------|--------|
| 얼음 (기본) | 하늘색 | 빙글이 본체와 동일한 톤 |
| 포근한 눈 | 화이트 | 포근하고 부드러운 겨울 |
| 민트 힐링 | 파스텔 그린 | 웰니스·회복 무드 |
| 베리 라벤더 | 파스텔 퍼플 | 아늑하고 몽글몽글 |
| 코지 우드 | 웜 베이지 | 스토리 속 카페 감성 |
| 동굴 슬레이트 | 쿨 그레이 | 얼음동굴과 어울리는 톤 |

- **바닥**: isometric 다이아몬드 타일(`assets/decor/floors/*.svg`) — 냉장고 하단 바닥 스트립에 타일링
- **벽**: 베벨 + 반짝임 포인트가 있는 얼음 블록 패널(`assets/decor/walls/*.svg`) — 냉장고 내부 벽 전체에 타일링
- 선택은 즉시 미리보기에 반영되고 `localStorage`(`bingle_decor_wall`, `bingle_decor_floor`)에 저장되어 새로고침해도 유지
- "↺ 기본값으로" 버튼으로 언제든 초기 얼음 테마로 복원 가능
- 벽·바닥 SVG는 `scripts/gen_decor_tiles.py`로 생성 — 팔레트만 바꾸면 새 테마를 쉽게 추가할 수 있음

## 앞으로 추가할 것

- 냉장고 꾸미기에 가구·조명·스티커 등 오브젝트 아이템 추가 (`#decor-layer`가 배치 레이어로 준비되어 있음).
  선반 위 소품(`.prop[data-slot]`)도 교체 가능한 슬롯으로 준비됨
- 산책 궤적 드로잉, 감사 편지 작성, V-log 등 미션별 상세 화면

## 캐릭터 & 아트 스타일

빙글이는 **2D 픽셀 아트 스프라이트**(256×256, 투명 배경)로 그려진 얼음 큐브 캐릭터입니다.

| 포즈 | 파일 | 쓰임 |
|------|------|------|
| 서기 | `assets/binglee/standing.png` | 메인 무대, 동굴 앞, 아바타, 산책/운동 재현 |
| 앉기 | `assets/binglee/sitting.png` | 영화·음악·식사 등 휴식 활동 재현 |
| 노트 필기 | `assets/binglee/working.png` | 일·독서·요리 활동 재현 (V-log) |

- `image-rendering: pixelated`로 픽셀이 선명하게 유지되고, CSS `hue-rotate` 필터로
  친구마다 빙글이 색이 달라집니다 (유림=핑크, 규빈=민트, 준호=웜톤, 소희=퍼플)
- UI도 픽셀 감성에 맞춰 **Galmuri 픽셀 폰트**(제목 Galmuri11 / 본문 Galmuri9)와
  각진 오프셋 그림자(`3px 3px 0`)를 사용합니다
- 8방향 회전 원본은 `bingle_motion` 에셋 폴더에 있으며, 산책 기능 구현 시
  `standing-east/west.png`를 걷기 방향 전환에 사용할 수 있습니다

## 실행

정적 페이지이므로 `index.html`을 브라우저로 열면 됩니다.

```
bingle_mainpage/
├── index.html
├── css/style.css
├── js/app.js
├── scripts/gen_decor_tiles.py  ← 냉장고 꾸미기 타일 생성 스크립트
└── assets/
    ├── binglee/   ← 픽셀 스프라이트 (서기·앉기·필기)
    └── decor/
        ├── floors/  ← 바닥 타일 6종 (SVG)
        └── walls/   ← 벽 타일 6종 (SVG)
```

## 기술

Vanilla HTML / CSS / JavaScript · Galmuri 픽셀 폰트 + Google Fonts (Jua, Gowun Dodum)
